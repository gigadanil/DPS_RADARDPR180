# MONOLITH 180 — Рация (Push‑to‑Talk)

Минимальный Node.js сервер + веб‑клиент (Socket.io + Multer).

## Запуск
1) В папке `ptt-server/`:
- `npm install`
- `npm run start`

2) Открой в браузере:
- `http://localhost:3030/`

## Как это работает
- Клиент держит кнопку → пишет звук (MediaRecorder)
- Отпустил / прошло 7 секунд → запись останавливается и загружается на `/ptt/upload` (Multer)
- Сервер сразу рассылает событие всем: `{ id, speakerName, url }`
- Клиенты перед воспроизведением говорят: "На связи <имя>" через `speechSynthesis`

## Ограничения (важно)
- Wake Lock помогает только пока вкладка видима; при заблокированном экране Android веб‑страница может быть приостановлена системой.
- В Telegram WebApp и в некоторых WebView фоновые ограничения могут быть сильнее.
